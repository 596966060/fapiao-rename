【一键部署 - 3分钟完成】

你现在有部署包：replit_deploy.zip

================================
最简单的方法 - 直接从GitHub导入
================================

1️⃣ 打开 https://replit.com

2️⃣ 找到你的旧 invoice 项目
   右上角菜单 (⋯) → Settings → 向下滚动 → Delete
   (或在项目卡片上右键 → Delete)

3️⃣ 点击左上角 "+ Create"

4️⃣ 寻找 "Import from GitHub" 选项
   (如果没有，点 "Templates" 或向下翻)

5️⃣ 点击 "Import from GitHub"
   会跳转到授权页面
   点 "Authorize Replit" 完成授权

6️⃣ 授权后，搜索框输入：
   fapiao-rename
   或完整地址：
   https://github.com/596966060/fapiao-rename

7️⃣ 点击搜索结果中的 fapiao-rename

8️⃣ 点击蓝色 "Import" 按钮
   Replit 开始下载所有文件
   等待1分钟左右

9️⃣ 导入完成，项目自动打开
   左边是文件列表，中间是编辑器，右边有 Run 按钮

🔟 点击右上角的 "Run" 按钮
   应用启动，右下角显示日志
   等待看到：✅ OCR 模型已准备好
   需要1-2分钟

1️⃣1️⃣ 等待完成后，右边会显示 "Webview" 区域
   有个网址（像 https://invoice-tool.replit.dev）
   点击网址打开应用

1️⃣2️⃣ 上传你的发票图片测试
   用鼠标拖拽或点击上传
   选择 screenshot-20260511-160543.png

1️⃣3️⃣ 等几秒钟，应该看到：
   ✓ 日期: 2026-05-07
   ✓ 发票号: 26324000000154072606
   ✓ 金额: 6.31 元

   (而不是空白或 - 符号)

完成！✅

================================
如果导入失败
================================

【备选方案】手动上传文件

1. 解压 replit_deploy.zip
   (得到 replit_deploy_package 文件夹)

2. 在Replit中：
   - 点击 "+ New Repl"
   - 选择 Python
   - 项目名: invoice-tool
   - 点 Create

3. 删除默认的 main.py：
   - 左边文件列表中找 main.py
   - 右键 → Delete

4. 上传文件夹中的所有文件到Replit：
   - app.py
   - requirements.txt
   - Procfile
   - .gitignore
   - templates/ (整个文件夹)

5. 点击 Run

6. 等待1-2分钟OCR初始化

7. 点击右边的网址打开应用

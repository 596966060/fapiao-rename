错误处理

如果 CMD 窗口显示错误：

1. "Address already in use"
   → 关闭占用 5000 端口的程序

2. "Python not found"
   → 安装 Python 3.7+

3. "Module not found"
   → 依赖安装中，请等待

4. 其他错误
   → 查看 CMD 窗口信息
   → 记录错误反馈

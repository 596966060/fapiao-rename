  3. 授权 GitHub
  4. 点 New + → Web Service
  5. 选你的仓库 invoice-rename-tool
  6. 点 Connect
  7. 配置：
     - Name: invoice-rename-tool
     - Runtime: Python 3
     - Build command: pip install -r requirements.txt
     - Start command: gunicorn app:app --timeout 120 --workers 1
  8. 点 Create Web Service
  9. 等 3-5 分钟...
  10. 看到 ✓ Live 就完成了
  11. 点右上角的链接，就是你的公开网址！
